# README — Shab Notepad

## What the project does

Shab Notepad is a modern, feature-rich text editor for Windows with advanced capabilities for developers and power users. It provides syntax highlighting, language detection, and a suite of built-in tools for productivity.

Key features include:
- Syntax highlighting for multiple programming languages (C#, C++, Java, JavaScript, Python, HTML, CSS, SQL, JSON)
- Language detection based on file extension
- Advanced text editing features (undo/redo, cut/copy/paste, find/replace, go to line)
- Word wrap and line numbering
- Dark and light theme support
- Font and encoding customization
- Built-in calculator with advanced expression evaluation
- Unit converter supporting 14 categories (length, weight, temperature, area, volume, data, time, speed, pressure, energy, power, number base, color, chemistry)
- Network converter with 16 network-related tools (IPv4/IPv6 conversion, CIDR calculations, nslookup, ping, tracert, etc.)
- Cryptographic hash calculators (MD5, SHA-1, SHA-256, SHA-512)
- ASCII/UTF-8 conversion tools
- Context menu options for quick actions (search in Google/ChatGPT/Grok, calculate, run in CMD)
- Recent files history and print capabilities
- Always on top and view in browser features

## Why the project is useful

Shab Notepad is designed to be a lightweight but powerful alternative to standard text editors. It combines the simplicity of a basic notepad with advanced features for developers and power users, making it suitable for:

- Writing and editing code with syntax highlighting
- Taking notes and writing documentation
- Calculating and converting units
- Network troubleshooting and calculations
- Cryptographic hash generation
- Quick access to search and calculation tools
- Everyday text editing tasks

## How users can get started with the project

### Requirements
- .NET 8.0 Runtime
- Windows operating system (x64)

### Installation
1. Download the latest release from the GitHub releases page.
2. Extract the contents of the zip file.
3. Run the `ShabNotepad.exe` executable.

### Usage

#### Basic Editing
1. **New File**: Click File → New or use Ctrl+N
2. **Open File**: Click File → Open or use Ctrl+O
3. **Save File**: Click File → Save or use Ctrl+S
4. **Copy/Cut/Paste**: Use toolbar buttons or Ctrl+C/Ctrl+X/Ctrl+V
5. **Find/Replace**: Click Edit → Find or use Ctrl+F

#### Advanced Features
1. **Syntax Highlighting**: Opens automatically based on file extension
2. **Language Detection**: Click View → Detect Language
3. **Calculator**: Select text and right-click → Calculate or use Tools → Calculator
4. **Unit Conversion**: Select text and right-click → Unit Converter or use Tools → Unit Converter
5. **Network Tools**: Select text and right-click → Network Converter or use Tools → Network Converter
6. **Hash Calculation**: Select text and right-click → Hash or use Tools → Hash
7. **Run in CMD**: Select text and right-click → Run in CMD or use Tools → Run in CMD

#### Viewing Options
1. **Dark/Light Theme**: Click View → Theme → Dark or Light
2. **Word Wrap**: Click View → Word Wrap
3. **Line Numbers**: Click View → Show Line Numbers
4. **Zoom**: Use Ctrl+Mouse Wheel or View → Zoom In/Out
